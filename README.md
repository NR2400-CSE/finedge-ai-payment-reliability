# FinEdge AI – Digital Payment Reliability Engine

## 🚀 Overview
AI-powered real-time monitoring system for ATM & digital payment failures.

## 🧠 Features
- Real-time ATM log streaming
- AI-based risk classification
- Failure rate analytics
- Web dashboard
- Socket-based live updates

## 🛠 Tech Stack
Frontend: React (Vite)
Backend: Node.js + Express
Realtime: Socket.io

## ▶ Run Locally

### Backend
cd server
npm install
node index.js

### Frontend
cd client
npm install
npm run dev

Server runs on: http://localhost:5000  
Frontend runs on: http://localhost:5173

## 🎯 Problem Solved
Reduces MTTR and detects ATM failures proactively.
